搬家具规则：

1.家具分不同的类型，并占用不同的面积
2.输出家具信息时，显示家具的类型和家具占用的面积 ――――――――――――――――――――――――――――――――――――――――――――――――――――――
3.房子有自己的地址和占用的面积
4.房子可以添加家具，如果房子的剩余面积可以容纳家具，则提示家具添加成功；否则提示添加失败
5.输出房子信息时，可以显示房子的地址、占地面积、剩余面积


class 家具：  # 定义一个家具类

	def 家具属性 初始值
		家具的类型
		家具的占用面积
	def str
		return  "房子的类型是%s,家具的占用面积%d"



class 房子：   # 定义一个房子类
	def 房子属性
		房子地址
		房子初始面积  =  总面积
	def str
		return "房子的地址%s,占用面积%d,剩余面积%d"

	def 添加家具   # 定义一个方法
		
		if 房子的面积 >= 家具占用面积
			print("家具添加成功"）
			房子剩余面积 = 房子初始面积 - 家具占用面积
		else:
		   	print("家具添加失败")
